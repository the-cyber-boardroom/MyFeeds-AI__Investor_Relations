## Key Success Factors from Top Angel/VC Pitches
Based on analysis of successful raises from companies like Airbnb, Uber, Front, Buffer, and Canva:

### Critical Design Principles
1. **Hook in 7 seconds**: The cover slide must instantly communicate the value proposition
2. **Story over data**: Lead with narrative, support with metrics
3. **Visual > Text**: Use infographics, icons, and diagrams (80% visual, 20% text)
4. **One message per slide**: Each slide should have ONE clear takeaway
5. **Progressive disclosure**: Build complexity gradually
6. **Emotional connection**: Make investors feel the pain before showing the solution
7. **Social proof early**: Credibility markers on every slide
8. **Clear ask**: Specific amount, use of funds, and expected outcomes

# PART 4: DESIGN SPECIFICATIONS

## Visual Design System

### Typography
**Fonts**:
- Headlines: Inter Bold (clean, modern)
- Body: Inter Regular  
- Accent: Fira Code (for technical elements)

### Color Usage
- **Primary (Navy)**: Headers, important text
- **Accent (Blue)**: CTAs, highlights, links
- **Success (Green)**: Positive metrics, checkmarks
- **Warning (Orange)**: Important callouts
- **Neutral (Grays)**: Body text, supporting info

### Iconography Style
- Use outlined icons (not filled)
- Consistent 2px stroke width
- Tech-forward but approachable
- Consider Lucide or Tabler icon sets

### Data Visualization
- Clean, minimal charts
- Always label axes
- Use color purposefully
- Include data sources

### Layout Principles
1. **White space is crucial** - don't overcrowd
2. **Visual hierarchy** - guide the eye
3. **Consistent alignment** - use grid system
4. **Progressive disclosure** - build complexity
5. **Mobile-friendly** - readable at small sizes

## Animation Guidelines (for digital delivery)
- Subtle slide transitions (fade/push)
- Data points appearing sequentially
- Graphs building over 2-3 seconds
- No distracting animations
- Focus on content, not effects
